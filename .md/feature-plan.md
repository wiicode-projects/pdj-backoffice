Once a user is connected to the backoffice check its role and store it. 
It will define what the user sees as menu or view. 

Admin: 
    - Dashboard 
        - Number of user 
        - Number of restaurant
        - Number of active subscription for restaurant 
        - Number of active subscription for user 
    - Menu: 
        - Abonnements (list and create new plan)
        - Utilisateurs (list of users not user linked to restaurant )
        - Restaurants (list of restaurant)

Restaurant: 
    - Dashboard
        - Number of dishes 
        - Number of view 
        - Number of click
    - Menu 
        - Dishes 
        - Menus 
        - Geo location
